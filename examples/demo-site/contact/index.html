About This Form

This form demonstrates modern CSS form validation with no JavaScript required:

  • Auto-generated asterisks: Required fields automatically show a red asterisk via CSS
  • Real-time validation: Fields turn green when valid, red when invalid (only after you interact with them) by relying on the user-valid and user-invalid pseudo-classes in CSS.
  • Error messages: Helpful messages appear automatically for invalid fields
  • Focus states: Notice the blue ring when you focus on inputs, and green/red rings for validation states
  • Theme support: All colors adapt to your selected theme (light/dark/auto)
