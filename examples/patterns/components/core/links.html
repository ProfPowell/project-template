Link Component

Text links with semantic variants. Uses data-link attribute.

Variants

Default Link

Standard inline links with underline. This is a paragraph with a default link in the middle of the text.

Navigation Link

Muted by default, emphasized on hover. Used in navigation contexts.

Subtle Link

Low emphasis links that reveal on hover. Good for secondary actions.

Accessibility

Current Page Indicator

Use aria-current="page" to indicate the current page in navigation.

External Links

Always include rel="noopener" for security when using target="_blank".

Descriptive Text

Avoid generic link text like "click here". Instead, use descriptive text that makes sense out of context.

Usage

<!-- Default inline link -->
<a href="/page" data-link>Learn more</a>

<!-- Navigation link -->
<a href="/about" data-link="nav">About</a>

<!-- Current page in navigation -->
<a href="/products" data-link="nav" aria-current="page">Products</a>

<!-- Subtle link -->
<a href="/details" data-link="subtle">View details</a>

<!-- External link -->
<a href="https://example.com" data-link="external" target="_blank" rel="noopener">
  External Resource
</a>
